Get Started

Mint a token, make a request, and persist data—no registrations, credit cards, or long-term contracts.

# 1. create a token (100 free operations)
curl https://token.echovalue.dev -d 'token=new'

# 2. write a value
curl https://api.echovalue.dev/default/hello \
  -H 'x-token: mytoken' \
  -d 'world'

# 3. read it back
curl https://api.echovalue.dev/default/hello \
  -H 'x-token: mytoken'
Tokens are scoped per project and can be rotated at any time. Credits never expire.
