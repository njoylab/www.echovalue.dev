EchoValue API

EchoValue is a lightweight and easy-to-use API designed for freelancers and developers working on small projects. It serves as a simple key-value database system, specifically tailored to store and retrieve key-value data without the need for complex server platforms.

Ideal for serverless applications, rapid prototyping, handling metrics, and much more.

With EchoValue, developers can conveniently store and access key-value pairs without worrying about the overhead of implementing a full-fledged database or cache system.
The API is built to be fast, efficient, and reliable, making it ideal for small-scale applications where simplicity and ease of use are paramount.

Get Started

Start for Free. Every new token creation gets 100 free operations.
Each API call costs 1 operation.
No Registration, No Credit card, No Subscriptions, No Expiration, No Hidden Costs 

// request a new token
curl 'https://token.echovalue.dev' \
-d 'token=new'

// save "world" under "hello" key
curl 'https://api.echovalue/default/hello' \
-H 'x-token: mytoken' \
-d 'world'

// get the value under "hello" key
curl 'https://api.echovalue/default/hello' \
-H 'x-token: mytoken'
% world
